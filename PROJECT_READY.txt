  🚀 HOW TO RUN
═══════════════════════════════════════════════════════════════════════════════

▶️  QUICKEST METHOD (Windows):
    1. Double-click: start.bat
    2. Wait for both windows to open
    3. Frontend opens at: http://127.0.0.1:8501
    4. Backend runs on: http://127.0.0.1:8000

▶️  MANUAL METHOD (Windows Terminal):
    Terminal 1:
    > venv\Scripts\activate.bat
    > python -m uvicorn api.main:app --host 127.0.0.1 --port 8000 --reload

    Terminal 2:
    > venv\Scripts\activate.bat  
    > streamlit run frontend/app.py --server.port 8501 --server.address 127.0.0.1

▶️  LINUX/MAC METHOD:
    $ chmod +x start.sh
    $ ./start.sh

  🛠️  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: "Port 8000 already in use"
Solution: 
  • Check if another FastAPI is running
  • Kill the process: taskkill /PID <PID> /F
  • Or use different port: --port 8001

Problem: "Model not found" error
Solution:
  • Verify: Processing files/NeuroSenti_Final_Submission/neuro_senti_best.h5
  • Check file path in api/main.py line 26
  • Run in project root directory

Problem: "Connection refused" on http://127.0.0.1:8000
Solution:
  • Backend not started - start it first
  • Wait 3-5 seconds after starting
  • Check terminal for any error messages

Problem: Streamlit page won't load
Solution:
  • Check port 8501 is not in use
  • Try: streamlit run frontend/app.py --server.port 8502
  • Update API_URL in frontend/app.py if needed
